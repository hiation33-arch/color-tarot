# 컬러 타로

색으로 보는 오늘의 타로. 마음에 끌리는 색을 골라 오늘의 메시지를 받아보세요.

## 바로 사용하기

👉 **[컬러 타로 열기](https://hiation33-arch.github.io/color-tarot)**

## 주요 기능

- 12가지 색상 카드 중 마음에 드는 **5장** 선택
- 5장 중 오늘의 색 **1장** 최종 선택
- 선택한 색에 맞는 키워드와 오늘의 메시지 표시
- 다시 뽑기 가능

## 사용 방법

1. 마음에 끌리는 색 카드를 5장 고릅니다
2. **다음 단계로** 버튼을 누릅니다
3. 5장 중 오늘의 색 하나를 최종 선택합니다
4. 오늘의 메시지를 확인합니다
